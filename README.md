# Civarchive TypeScript SDK

[![NPM version](https://img.shields.io/npm/v/civarchive.svg)](https://npmjs.org/package/civarchive)

This library provides convenient access to the Civarchive REST API from server-side
TypeScript or JavaScript.

## Installation

```sh
npm install civarchive
```

## Usage

```ts
import Civarchive from 'civarchive';

const client = new Civarchive({
  apiKey: process.env['CIVARCHIVE_API_KEY'],
});
```

## Development

This SDK's `src/` is generated by [opensteel](https://github.com/) from the committed
`openapi.yaml` + `stainless-config.yml`. The publishing wrapper (build scripts, configs,
and workflows) was scaffolded once with `opensteel init`.

- **Regenerate:** edit `openapi.yaml` / `stainless-config.yml` and push — the `Generate SDK`
  workflow opens a pull request with the new `src/`.
- **Release:** merge generated PRs with conventional-commit titles (`feat:` / `fix:`);
  release-please opens a `release: x.y.z` PR. Merging it publishes to npm.

Build locally with:

```sh
yarn install
yarn build
```
